# F407VGT6_DRIVER 引脚与接线说明

本工程当前用于 STM32F407VGT6 控制两片 19536 半桥组成的单相全桥，通过 SPWM 完成 DC-AC 变换。当前软件配置为 36 V 直流母线、20 kHz PWM 载波、20~100 Hz 可调输出频率和 18~25 Vrms 可调输出电压。

> 本文按当前源码和 `F407VGT6_DRIVER.ioc` 整理。接线前请先断开 36 V 功率电源，只保留调试器或限流逻辑电源检查 PWM。STM32 GPIO 和 ADC 均不耐受 5 V，任何进入 STM32 的模拟或数字信号必须限制在 0~3.3 V。

## 1. 全桥 SPWM 输出

两片 19536 分别驱动全桥的两个桥臂。每个桥臂的 HI、LI 必须使用同一组 TIM1 主/互补通道，不能交叉接到另一组通道。

| STM32 引脚 | 定时器功能 | 连接位置 | 说明 |
| --- | --- | --- | --- |
| PA7 | TIM1_CH1N | 第一片 19536 的 HI1 | 第一桥臂高管控制输入 |
| PE9 | TIM1_CH1 | 第一片 19536 的 LI1 | 第一桥臂低管控制输入，与 PA7 硬件互补 |
| PB0 | TIM1_CH2N | 第二片 19536 的 HI2 | 第二桥臂高管控制输入 |
| PE11 | TIM1_CH2 | 第二片 19536 的 LI2 | 第二桥臂低管控制输入，与 PB0 硬件互补 |

当前 TIM1 参数：

- PWM载波频率：20 kHz。
- 计数方式：向上计数，ARR=999。
- 两组主/互补输出均为高电平有效。
- 硬件死区：约1.25 us。
- 停止、故障或过流时通过关闭TIM1的MOE同时关断四路PWM。
- 第一、第二桥臂采用相反的正弦调制参考，负载电压取两个半桥中点之间的差分电压。

19536的逻辑地必须与STM32 GND可靠共地。STM32引脚只能连接19536的逻辑输入端，不得连接MOS管栅极、半桥中点或36 V母线。两个半桥中点分别作为全桥输出A、B，后接低通滤波器和负载。

## 2. 交流电压、电流采样

| STM32引脚 | ADC通道 | 外接模块 | 连接方式 | 当前软件用途 |
| --- | --- | --- | --- | --- |
| PA0 | ADC1_IN0 | TA1015-2M | 模块模拟输出OUT接PA0 | 输出电流瞬时采样、RMS显示及过流保护 |
| PA1 | ADC1_IN1 | TV1013-1H | 模块模拟输出OUT接PA1 | 输出电压瞬时采样、RMS显示及MODE1闭环反馈 |
| PA2 | ADC1_IN2 | 旧DC-DC输入电压采样 | 原电压采样Vout接PA2 | 仅由旧DC-DC功能保留，SPWM控制不使用 |
| PA3 | ADC1_IN3 | 旧DC-DC辅助电压采样 | 原电压采样Vout接PA3 | 仅由旧DC-DC功能保留，SPWM控制不使用 |

采样注意事项：

- TA1015-2M和TV1013-1H的模拟输出中心电压约为1.69 V，交流量叠加在该偏置上。
- 模块GND、STM32模拟地和19536逻辑地需要共地，但功率回流与采样地应采用单点或合理的星形接地，避免MOS开关电流流经ADC地线。
- 接入PA0、PA1前，必须先用示波器确认模块输出在所有工况下都位于0~3.3 V，严禁负压或超过3.3 V直接进入ADC。
- PA0/PA1附近建议加入串联限流电阻及3.3 V钳位保护；是否增加RC滤波应根据采样带宽和实测开关尖峰确定。
- 当前ADC由TIM3以10 kHz触发。PA0、PA1、PA2、PA3和内部参考电压组成5通道DMA循环采样。
- 当前软件中的电压比例系数为62.2222 V/V，电流比例系数为8.3333 A/V。最终系数必须用标准表在实际负载下校准。

TV1013-1H的交流输入应跨接在滤波后的负载两端，而不是任一桥臂中点对地。TA1015-2M如用于验收输出电流，应串接或穿芯于滤波器之后、负载之前；若安装在桥臂或滤波电容之前，测得的会包含开关纹波及无功电流，不能直接代表负载RMS电流。

## 3. OLED屏接口

当前OLED使用SPI3单向发送，不使用MISO。

| STM32引脚 | OLED引脚 | 功能 |
| --- | --- | --- |
| PC10 | SCK/CLK | SPI3时钟 |
| PC12 | MOSI/DIN | SPI3数据输出 |
| PC11 | DC | 命令/数据选择 |
| PD0 | RST/RES | OLED复位 |
| PD1 | CS | OLED片选 |
| 3.3V | VCC | OLED电源，须确认屏模块支持3.3 V |
| GND | GND | 公共地 |

SPI3当前工作在主机模式、CPOL=0、CPHA=第1边沿，软件片选，时钟约20 Mbit/s。

## 4. 4×4矩阵键盘

### 4.1 GPIO连接

| 矩阵方向 | STM32引脚 | 软件名称 | GPIO配置 |
| --- | --- | --- | --- |
| 第1行（最上） | PD3 | H1 | 推挽输出、下拉 |
| 第2行 | PD4 | H2 | 推挽输出、下拉 |
| 第3行 | PD7 | H3 | 推挽输出、下拉 |
| 第4行（最下） | PD10 | H4 | 推挽输出、下拉 |
| 第1列（最左） | PD11 | V1 | 输入、下拉 |
| 第2列 | PE1 | V2 | 输入、下拉 |
| 第3列 | PE3 | V3 | 输入、下拉 |
| 第4列（最右） | PE4 | V4 | 输入、下拉 |

实体键盘排列：

```text
1  2  3  A
4  5  6  B
7  8  9  C
*  0  #  D
```

代码内部扫描码不是1~16按行排列，当前实体键与扫描码对应如下：

```text
1(2)   2(1)   3(3)   A(4)
4(6)   5(5)   6(7)   B(8)
7(10)  8(9)   9(11)  C(12)
*(14)  0(13)  #(15)  D(16)
```

### 4.2 当前SPWM模式下的按键功能

| 实体按键 | 功能 |
| --- | --- |
| 1 | 启动MODE0开环SPWM |
| 2 | 启动MODE1 PID闭环SPWM |
| 3 | 进入输出电压设置页 |
| * | 进入输出频率设置页 |
| A | 设置页面中数值增加1 |
| B | 设置页面中数值减少1 |
| # | 确认设置并启动当前选择的模式 |
| D | 停止输出、取消设置或清除故障 |
| C | 当前SPWM界面中未分配功能，预留 |

电压设置范围为18~25 Vrms，默认18 Vrms；频率设置范围为20~100 Hz，默认50 Hz。进入设置页面时会先关闭四路PWM，按 `#` 后重新启动。

## 6. 上电前检查顺序

1. 断开36 V母线和MOS管功率级，只给STM32、OLED及驱动逻辑供电。
2. 检查PA7/PE9和PB0/PE11是否分别构成两组互补PWM，确认死区存在且同一桥臂无重叠导通。
3. 执行停止操作，确认四路PWM均进入无效状态。
4. 用信号源或低压交流验证PA0、PA1始终处于0~3.3 V，并检查约1.69 V的中心偏置。
5. 接入限流的低压直流母线，先使用MODE0和较低设定值验证两个半桥中点及差分输出。
6. 确认滤波后波形、采样显示和保护均正常后，再逐步升至36 V并增加负载。

严禁将普通台式示波器的两个地夹分别接到两个半桥中点。地夹通常与保护地相连，会直接短路桥臂。测量全桥差分输出应使用高压差分探头，或用两个对地通道配合数学相减，并确保示波器及被测系统的接地方式安全。
